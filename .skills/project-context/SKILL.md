# Project Context — micro-stop-detection

## Project summary
University AI course project. System to detect, classify and predict industrial micro-stops (1-10 second interruptions) in production lines using manually implemented neural networks with numpy only.

## Team responsibilities

| Member | Role |
|--------|------|
| Aaly   | Data preprocessing + visualizations |
| **Xavi (Xavier)** | **Model architecture — LSTM, Attention, PredictionHead** |
| Jesus  | Detection module (binary) + Classification module (multiclass) |
| Pedro  | Dataset script + notebook integration |
| Yess   | Results visualizations + impact simulation |
| Victor | Documentation + docstrings |

## Dataset: `industrial_dataset.csv`
- Shape: (20000, 6)
- Columns: `vibration, current, speed, temperature, micro_stop, cause`
- micro_stop: 0=normal, 1=micro-parada (~0.3% of rows)
- cause: 0=normal, 1=falta_material, 2=desalineacion, 3=error_humano, 4=fallo_mecanico
- Contains: ~1% NaN per sensor column, temporal lag between cause and event
- Generated by: `notebook/dataset.ipynb` (DO NOT modify generator logic)

## Architecture

```
Input: (batch, window_size=20, features=4)
         |
    [LSTMCell -- Xavi]
    all_h: (batch, 20, hidden=32)
         |
    [Attention -- Xavi]
    context: (batch, 32)
         |
    +------------------------+
    |  DetectionHead (Jesus) | -> sigmoid -> prob_detect
    |  ClassificationHead    | -> softmax -> prob_cause (4 classes)
    |  PredictionHead (Xavi) | -> relu -> steps_to_next_stop
    +------------------------+
```

## Key integration point
`extract_context(X_batch)` is the bridge between Xavi's LSTM and Jesus's heads:
- Input: `(batch, 20, 4)`
- Output: `(batch, 32)` — same contract as the `simple_rnn_forward` placeholder

## Evaluation targets
- Recall (detection) > 0.75 — this is the most critical metric
- F1-score reported for all modules
- Anticipation time: average steps before micro-stop that system detects it
- Downtime reduction simulation based on anticipation time

## Hard constraints
- NO Keras/PyTorch/TensorFlow
- numpy only for all math
- sklearn ONLY for metric calculation
- Backpropagation must be manual (BPTT for LSTM)
- Gradient clipping required
